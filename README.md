I was tasked to create the frame work for a messenger app. The main purpose of this code was to collect, process and disply the data the user provided to the questions corresponding to a format relating to a report. There were 3 main plain points that the user is meant to fill in, that were the username, password, and phone number. Each variable had their own specific requirements that needed the user to fill in while being aware of the instructions being layed out to them if they made a mistake in the process. For when said mistakes do happen theyh will will taken through a [ while loop ] that will keep on repeating the exact same questions until the conditions are met.

The username is made to collect the user name under the conditions that it was meant to contain a length of five characters and a special charcter "_". The password; 8 characters long with special characters with any capital letter. The user's phone number a variabale with 12 long character and its first two should be written by the users country phone number code, e.g. South African phone number is universal as they all start with +27.
